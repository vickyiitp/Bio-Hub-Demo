# 🌟 Bio-Hub-Demo: The Ultimate Link Management for Creators

Bio-Hub is a premium, open-source alternative to Linktree, designed for digital creators who demand a high-end, customizable, and professional online presence. 

This project serves as a **live demo** of a modern link management tool built with performance and aesthetics at its core.

---

## 🚀 Why Bio-Hub? (Key Benefits for Digital Creators)

In the digital-first economy, your "link in bio" is your storefront. Bio-Hub empowers creators by offering:

- **💎 Premium Aesthetics**: Say goodbye to generic layouts. Bio-Hub features modern design systems like Cyberpunk, Glassmorphism, and Brutalist themes.
- **⚡ Seamless User Experience**: Experience fluid transitions and responsive layouts that keep your audience engaged.
- **📊 Actionable Insights**: Built-in mock analytics show you which links are performing best, helping you refine your strategy.
- **🛠️ Total Control**: Manage your links with an intuitive drag-and-drop interface. Real-time mobile preview ensures you know exactly what your fans see.
- **📈 Professional Branding**: Present your portfolio, social media, and products in a space that feels like a custom-coded website, not just another profile.

---

## ✨ Core Features

- **Split-Screen Builder**: Manage your links on the left, see the results instantly on the right.
- **Dynamic Themes**: Instantly switch between multiple high-end design presets.
- **Drag-and-Drop Editor**: Reorder your top links effortlessly using `@dnd-kit`.
- **Live Mobile Preview**: A gorgeous, interactive mobile frame powered by `framer-motion`.
- **Simulated Analytics**: Track clicks and engagement through a sleek analytics dashboard.
- **Local Persistence**: All changes are saved to your browser automatically, no database setup required for this demo.

---

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Drag & Drop**: dnd-kit
- **Language**: TypeScript

## 🏁 Getting Started

To run this demo locally:

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/vickyiitp/Bio-Hub-Demo.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Open in your browser**: Navigate to `http://localhost:3000`.
